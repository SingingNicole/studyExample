* 자바스크립트의 객체와 배열
객체(Object): 속성(property, 이름이 있는 값)을 모아 둔 공간
			{이름 : 값, 이름 : 값, ...};의 형식으로 선언
			let person = {name:"javakim", age:20, job:"student",
				study: function(){ // 함수도 객체로 선언 가능하다.
					alert("공부")
					;}
				};
배열(Array): 번호가 붙은 값들을 모아 둔 공간
			let fruits = ["사과", "바나나", "자두"]; => 0, 1, 2, ... 순으로 번호에 저장
			
			* 객체 선언, string, date, 배열 등 기억해두기.
