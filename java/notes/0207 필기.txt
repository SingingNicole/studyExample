메서드 선언
[접근제어자] 리턴타입 메서드명(타입 변수명, 타입 변수명...){ // 메서드의 매개변수도 지역변수이다(메서드가 끝나면 종료됨)
	수행문;
	return 값; // 한 개의 값만 리턴할 수 있다.
	}
	
호출
리턴타입 변수 = 참조변수.메서드명(값, 값, ...);

메서드 매개변수
메서드 종료 - 지역변수

기본형 -> 작은 값에서 큰 값으로 이동할 경우 자동 변환됨.
다른 메서드에 있는 매개변수는 기능이 같아도 아무런 상관이 없다. 
-> 메서드가 종료되면 지역변수는 소멸하고 호출된 지점으로 돌아간다.
	*메서드가 끝나는 경우
		1. return문이 실행되었을 때
		2. 마지막 블럭에 도달했을 때

static이 붙어 있는 멤버는 static 메서드만 사용 가능하다.
instance 변수는 static 변수, 같은 시점의 instance 변수를 사용 가능하다.

생성자 선언
클래스명 () {}
클래스명(인스턴스 변수에 맞게 지역변수 선언){
	인스턴스변수 = 지역변수;(주로 값 저장)
}

호출 - 객체 생성 시 'new' 뒤에서 항상 생성자 호출
	new 생성자(값...); // 매개변수에 맞추어 값 지정
	-> 객체를 생성할 때만 사용한다.
	
생성자가 하는 일 : 인스턴스 변수에 값 저장(초기화)

this : 자기 자신
this() : 생성자 호출. 같은 클래스 내의 생성자 호출(매개변수 맞춰야 함)
this. : 참조변수. 현재 생성한 객체의 주소값을 저장하고 있음.
		인스턴스변수와 생성자 내 매개변수 이름이 같은 경우
		객체에 있는 인스턴스변수를 찾아준다.
