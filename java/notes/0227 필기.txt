git - 형상관리
소프트웨어의 변경사항을 체계적으로 추적하고 통제하는 것(버전 관리)
형상 관리는 일반적인 단순 버전관리 기반의 소프트웨어 운용을 좀 더 포괄적인 학술 분야의 형태로 넓히는 근간

일반적으로 형상 항목(영어: Configuration Item)이라는 형태로 작업 산출물을 선정
형상 항목 간의 변경 사항 추적과 통제 정책을 수립하고 관리

clone 후
개인 branch 생성 -> 커밋 -> push 원격 개인 브랜치 -> pull request 개인 -> main(merge)

main 브랜치로 이동
원격 main -> 로컬 main: pull
로컬 main - 개인 브랜치: merge
