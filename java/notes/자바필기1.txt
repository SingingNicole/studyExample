자바의 특징

1. 객체지향(OOP : Object Oreinted Programming)
절차 지향 프로그래밍 - 코드 작성 순서대로 실행
객체 지향 : 내가 필요로 하는 것을 미리 객체로 형성
객체들을 조립해서 프로그램을 실행시킨다.
2. garbage collection(GC)
필요 없어진 데이터(객체)를 자동으로 삭제. 메모리 관리를 자동으로
쓰레기 통에 모아 버려 줌. 메모리 관리를 할 필요 없이 사용할 수 없는 변수나 객체를 처리.
3. 플랫폼 독립성
4. Multi Thread
프로세스 : 운영체제 상 실행 중인 프로그램
스레드 : 프로그램 내 최소 작업 단위
멀티 스레드 : 한 프로그램에서 여러 작업을 동시에 (번갈아)진행 가능
