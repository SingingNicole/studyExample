class : 객체(실제 메모리상에 존재하는 것)를 생성한다.

변수
생성자
메서드 - for, if, while => 메서드 안에 존재

클래스 멤버
1. 변수
2. 생성자
3. 메서드 - for, if, while

메인 메서드 - 실행용 클래스
나머지 클래스 - 객체를 활용하여 작업

클래스 선언 : 접근제어자 제어자 class 클래스명{ // 헤드
}
클래스 안에 모두 변수가 들어가 있어야 함.

스마트폰 - 기종(model), 색상(color), 통신사(com)...
	// 수신한다, 송신한다.
class Phone{ // 변수명으로 활용
	변수타입 + 변수명;
	String model;
	String color;
	String com;
}

Phone p -> Phone 클래스로 만든 객체의 주소값만 저장 가능.
Phone p = new Phone() -> 힙(heap)에 실제 메모리에 존재하는 데이터를 입력한다.

배열 : 같은 타입의 값을 여러개 모아서 저장하는 공간
